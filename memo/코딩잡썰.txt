<코딩잡썰>

- 결국, 어떤 문제를 해결하기 위해.. 자동적으로 일을 하기 위해, 컴퓨터라는 단순하지만 상당히 빠른 노예에게 일을 시키기 위해 프로그래밍(코딩)을 하게 된다.

- 클래스를 이용한다기 보단, 메소드를 이용한다고 보면 된다. 메소드를 활용하는 게 JAVA 프로그래밍의 핵심이다.
- 그렇다면 어떻게 메소드를 활용할 것인가?
메소드를 활용하려면..

1. 일단, 다른 패키지에 속한다면, import를 해야 할 것. 그냥 길게 써도 되고.
2. 해당 클래스의 객체를 생성해야 할 것(static 선언이 돼 있는 경우 바로 쓸 수 있음).
3. 원하는 메소드를 호출해서 사용.

- 생성자가 다양하다는 건 무엇을 의미하는가? 해당 인자값에 대응하는 클래스는 생성?

- 자주 사용되는 메소드를 정리할 것!
